# Berita App

Aplikasi mobile berita menggunakan Expo + React Native.

## Features
- Fetch API berita
- Search berita
- Bookmark
- Infinite scroll

## Tech Stack
- Expo
- React Query
- Axios

## Setup

1. Clone repo
2. Install dependencies
3. Buat file .env
4. Jalankan app
